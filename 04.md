# Clase 4
## Más Racket:
- If, Elif, ..., Else:
```
(cond [cond_1 resultado_1] [cond_2 resultado_2] ... [else resultado_n])
```
- Comentarios:
```
; línea de comentario
;; línea de comentario importante
#| bloque de comentario |#
#; comentariar un bloque de código
```
- Lambda (funciones anónimas):
```
(lambda (param_1 param_2 ...) resultado)
```
